Step 3: Setting up a Job

Previous | Contents | Next

We have defined regions in an image and saved it as a template,  now we need to set up a job to use this template.


  1. Click the Job tab.
  2. In the Job Name text field, type NEWJOB


  3. Right click in the job template field,  and select 'Set Job Template',  this will set the job template of this job to the template you are currently editing.


  4. To the right of the Image Path field,  select the folder button to search for the <sample pack path>/sample/images path.


  5. In the output path,  type OUT.  By the Job Name,  click the button with the picture of the disk. This will save this job.


  6. Click Execute... you should see a command line window open up. 


  7. After the command line window disappears (this should take a few minutes),  right click in the Job Name field and select 'Explore'.  This will open up the Job Directory in Explorer. 


  8. Note in this path,  you should see a NEWJOB.BAT file.  When you create a schedule,  this is the BAT file you will use run at some short interval.  Under the OUT path,  you will see the XML generated by this run.


  9. Go to Control Panel->Task Scheduler (on Vista) or Control Panel-> on vista.




  10. VISTA:  Create a new Task,  and set it to run every 2 minutes for 1 hour.  Set this module to execute to the NEWJOB.bat.  Enable this job.




  11. XP: Create a new Task,  and set it to run every 2 minutes for 1 hour.  Set this module to execute to the NEWJOB.bat. Enable this job.
